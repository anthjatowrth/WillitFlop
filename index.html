01 — Concept

Qu'est-ce que WillItFlop ?

🎮
La problématique
Dans l'industrie du jeu vidéo, estimer la viabilité commerciale d'un titre avant sa sortie est un défi majeur. Des studios indépendants aux grands éditeurs, de nombreux jeux échouent malgré des budgets conséquents — faute d'une analyse de marché rigoureuse et de données exploitables en amont.

WillItFlop répond à cette question : « Si ce jeu vidéo existait, aurait-il des chances de réussir commercialement ? »
💡
La solution
Une plateforme web où l'utilisateur décrit un jeu vidéo fictif (genre, plateformes, budget, univers…). Le système collecte des données réelles sur des jeux similaires, les analyse, et produit un score de viabilité commerciale accompagné d'un rapport détaillé.
🎯
La valeur ajoutée
Le projet combine collecte de données réelles (API gaming, web scraping), pipeline de traitement automatisé (Airflow + PostgreSQL) et IA générative pour produire une expérience à la fois analytique et immersive — jusqu'à générer une image et un synopsis du jeu fictif.
02 — Objectifs

Ce que le projet démontre

01
Maîtrise de la collecte de données — Interroger des APIs tierces (données gaming) et effectuer du web scraping pour enrichir une base de référence sur les jeux vidéo existants.
02
Pipeline de données automatisé — Orchestrer les flux de données avec Apache Airflow, stocker et interroger les données dans PostgreSQL, assurer la qualité et la cohérence des données.
03
Analyse & scoring — Croiser les caractéristiques du jeu fictif avec les données historiques du marché pour produire un score de viabilité commerciale argumenté.
04
Intégration de l'IA générative — Utiliser des APIs IA pour générer automatiquement une image de couverture et un synopsis du jeu fictif, rendant l'expérience utilisateur immersive.
05
Restitution claire et impactante — Présenter les résultats via un site web custom intégrant des visualisations PowerBI et une interface soignée à destination de l'utilisateur final.

05 — Intelligence Artificielle

Fonctionnalités IA / ML du projet

🖼️
Génération d'image
À partir de la description du jeu fictif, une API de génération d'image produit automatiquement une illustration de couverture stylisée et cohérente avec l'univers décrit.
✍️
Génération de texte
Un modèle de langage génère un synopsis narratif du jeu fictif ainsi qu'un résumé analytique de ses forces et faiblesses commerciales.
📊
Scoring IA
Croisement des données historiques du marché avec les caractéristiques du jeu pour produire un score de viabilité commerciale argumenté et contextualisé.

08 — Livrable

Résultat attendu

Le livrable final est un site web interactif permettant à n'importe quel utilisateur de décrire un jeu vidéo fictif et d'obtenir en retour :

• Une image de couverture générée par IA
• Un synopsis narratif du jeu
• Un score de viabilité commerciale sur 10
• Un rapport analytique comparant le jeu fictif au marché réel
• Des visualisations Python (Matplotlib / Seaborn) intégrées au site sur le positionnement marché

L'ensemble est hébergé sur un site web custom développé en HTML/CSS/JS, dans la charte graphique Cyberpunk Neon Light du projet.
